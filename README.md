# actions.nvim

## Installation

With [packer.nvim](https://github.com/wbthomason/packer.nvim):

```lua
use "lpoto/actions.nvim"
```

With [vim.plug](https://github.com/junegunn/vim-plug)

```vim
Plug "lpoto/actions.nvim"
```


##

See `:help actions.nvim` or [help](./doc/actions.txt).
